rFactor2 Semaphore DX11 plugin

Instalacao:
1 - Copiar SemaphoreDX11.dll para a pasta rFactor 2/Bin64/Plugins
2 - Activar o Plugin no menu do jogo e reiniciar o jogo

Funcionamento:
Quando o piloto se encotra na pitlane acendem 4 luzes amarelas

Quando o piloto se encontra na pista durante a contagem decrescente para o incicio da corrida
acendem luzes vermelhas que acompanham a acender das luzes do circuito.
O plugin acende sempre 1 ate 4 luzes, independentemente do numero de luzes no circuito em questao.
Quando as luzes do circuito se desligam, as luzes do plugin desligam-se todas em simultaneo.

Logging:
Este plugin produz um ficheiro de log com o nome "SemaphoreDX11.log" na pasta de instalacao do plugin.
Em caso de falha, enviar este ficheiro para o autor.
